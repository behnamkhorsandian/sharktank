## Github process

### 1- New Repo:
```
git init
git add .
git status
git commit -m “first commit”
git branch -M main
git remote add origin [https://repo.git]
git push -u origin main
```

### 2- Update Repo:
```
git status
git add .
git status
git commit -m "Vx.x.x - comment”
git push -u origin main
```

### 3- Restore to commit:
```
git reset --hard <commit-sha1-id>
git push origin HEAD --force
```
